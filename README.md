# myrender
一个学习的渲染器（来自tinyrender） 
本项目最初的原形来自于tinyrender  
https://github.com/ssloy/tinyrenderer/wiki  
不是太了解相关协议，因此没有写协议，遵循上述链接作者使用的要求。  
项目暂时期望：（按实现顺序  
1、实现线框绘制，支持obj等文件格式，输出tga、png等图片文件格式。  
1.1 输出tga文件  
1.2 绘制线条  
1.3 建立自己的数学库  
1.4 输入obj文件  
1.5 线框建模  
2、实现面元绘制，三角形面  
3、遮隐的画家、z等算法  
4、放大缩小  
5、视角相机移动  
6、阴影生成  
7、bling-phong  
8、绕轴旋转、位移  
9、半透明体构造  
10、流体生成  
...
